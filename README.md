# DriveEasy Car Rentals — Landing Page

A modern React + TypeScript landing page for DriveEasy, a fictional car rental company. Showcases the fleet, pricing, testimonials, and a contact form.

---

## Prerequisites

- Node.js >= 20
- npm >= 9

---

## Setup

```bash
git clone git@github.com:Offthearc/car_rental_landing_page.git
cd car_rental_landing_page
npm install
```

---

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

---

## Tests

```bash
npm test
```

---

## Lint & Format

```bash
npm run lint
npm run format:check
```

---

## Build

```bash
npm run build
```
